Introduction (1/4)

We study the usability of passwords. In this survey, you will be asked to create a password. Please behave as naturally as possible, that means, please answer intuitively. There are no correct or wrong answers; there are no “good” and “bad” passwords.

You should not use a password that you already use elsewhere, simply behave as if you were asked to choose a new password for your primary email account.

We will not store any data that can be traced back to you. Neither will we store your password. We are only going to collect some meta-information, e.g. the length of your password or the number of special characters it contains.

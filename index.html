Introduction (1/4)

We study the usability of passwords. In this survey, you will be asked to create a password. We are not going to store any personal data. However, please behave as naturally as possible, that means, please answer intuitively. There are no correct or wrong answers, that means there are no “good” and “bad” passwords. You should not use a password that you already use elsewhere, but one that you think is strong enough and that you could hypothetically remember easily.

# Emoji Password #

## Requirements ##
1. Bower (`$ npm install -g bower`)
2. xAMP Stack (Apache, MySQL, PHP)


## Setup ##
1. Run `$ bower install`
2. Create a file `php/connection.private.php` that contains the database connection settings. This file won't 
be committed.
